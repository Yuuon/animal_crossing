## 简介

datoucai是一个基于酷Q和Nonebot SDK的QQ机器人，为QQ群内的大头菜交易提供便利功能。

## 功能列表

- [x] 开房和排队
- [x] 价格储存与展示
- [x] 数据持久化（json）
- [ ] 数据持久化（数据库）
